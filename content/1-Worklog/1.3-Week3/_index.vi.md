### Mục tiêu tuần 3:

* Hiểu các khái niệm cơ bản về mạng (Networking) trên AWS.
* Tìm hiểu kiến trúc và các thành phần của Amazon Virtual Private Cloud (Amazon VPC).
* Thực hành cấu hình các tài nguyên mạng trên AWS.
* Hiểu cách các dịch vụ mạng của AWS giúp các tài nguyên giao tiếp an toàn với nhau.
* Nâng cao kỹ năng thực hành với Amazon VPC, Subnet, Route Table, Internet Gateway và Security Group.

### Kết quả đạt được tuần 3:

* Hiểu được các khái niệm cơ bản về mạng trên nền tảng AWS và vai trò của hệ thống mạng trong môi trường điện toán đám mây.

* Nắm được các thành phần chính của Amazon VPC, bao gồm:
  * Virtual Private Cloud (VPC)
  * Subnet
  * Route Table
  * Internet Gateway
  * Security Group
  * Network ACL

* Tạo và cấu hình thành công một Amazon VPC với Public Subnet và Private Subnet.

* Cấu hình Internet Gateway và Route Table để các tài nguyên trong VPC có thể kết nối với Internet theo đúng thiết kế.

* Hiểu được sự khác nhau giữa Security Group và Network ACL, đồng thời biết cách cấu hình các quy tắc bảo mật phù hợp.

* Thực hành kết nối Amazon EC2 trong VPC và kiểm tra khả năng giao tiếp giữa các tài nguyên.

* Nâng cao kỹ năng thiết kế, triển khai và cấu hình hạ tầng mạng cơ bản trên nền tảng AWS thông qua các bài thực hành.
