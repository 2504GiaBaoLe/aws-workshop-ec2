* Hiểu những kiến thức cơ bản về Serverless Computing trên AWS.

### Kết quả đạt được tuần 5:

* Hiểu được khái niệm Serverless Computing và những lợi ích của mô hình này trong việc phát triển các ứng dụng điện toán đám mây hiện đại. 
* Tìm hiểu cách AWS Lambda thực thi mã nguồn mà không cần quản lý máy chủ. 
* Tạo và cấu hình thành công AWS Lambda Function, bao gồm: 
  * Cấu hình Runtime 
  * Environment Variables 
  * IAM Execution Role 
  * CloudWatch Logs 
  
* Tìm hiểu cách cấu hình Lambda Trigger từ các dịch vụ: 
  * Amazon API Gateway 
  * Amazon S3 
  * Amazon EventBridge 
  * DynamoDB Streams 
  
* Xây dựng thành công REST API bằng Amazon API Gateway, bao gồm: 
  * Resource và Method 
  * Tích hợp với Lambda 
  * Deployment Stage 
  * Cấu hình CORS 
  
  * Tìm hiểu các kiến thức cơ bản về Amazon DynamoDB: 
    * Tạo bảng dữ liệu (Table) 
    * Partition Key 
    * Item và Attribute 
    * Các thao tác CRUD 
    * Query và Scan 
    
  * Kết nối AWS Lambda với DynamoDB thông qua AWS SDK để thực hiện các thao tác với dữ liệu. 
  
  * Xây dựng thành công REST API Backend theo mô hình Serverless với các chức năng: 
    * Tạo dữ liệu (POST) 
    * Lấy dữ liệu (GET) 
    * Cập nhật dữ liệu (PUT) 
    * Xóa dữ liệu (DELETE) 
    
  * Có được kinh nghiệm thực tế trong việc tích hợp API Gateway, Lambda và DynamoDB để xây dựng một ứng dụng Serverless hoàn chỉnh. 
  
  * Nâng cao hiểu biết về việc xây dựng các ứng dụng có khả năng mở rộng, hướng sự kiện và sử dụng các dịch vụ được quản lý hoàn toàn trên AWS.
